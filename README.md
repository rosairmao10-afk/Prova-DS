# Gerenciador de Tarefas

Aplicação web para gerenciamento de tarefas (CRUD), desenvolvida como projeto acadêmico da **ETEC**, em grupo.

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- PostgreSQL

## ✨ Funcionalidades

- Cadastrar novas tarefas (título, descrição e nível de urgência)
- Listar todas as tarefas cadastradas
- Editar tarefas existentes
- Excluir tarefas
- Feedback visual por meio de notificações (toasts)

## 🚀 Como rodar o projeto

1. Clone o repositório do GitHub
2. Configure a variável `DATABASE_URL` em `script_db.js` com a connection string do seu banco Neon.
3. Abra o arquivo `index.html` no navegador (ou utilize uma extensão como o Live Server).
